# TP – Test Driven Development (TDD) en Python

Ce projet est réalisé dans le cadre du module **R504 – Développement Guidé par les Tests (TDD)**.  
Toutes les étapes ont été développées avec la méthodologie **Test First** et versionnées avec **Git + GitHub Desktop**.

---

## Objectifs

✔ Utiliser le TDD :  
1. Écriture d’un test **qui échoue**  
2. Écriture du code **juste suffisant pour passer le test**  
3. **Refactorisation** sans casser les tests  
4. Commit Git à chaque étape

✔ Versionner proprement l’évolution du code sur **GitHub**  
✔ Implémenter deux exercices :  
- **Question 1 : FizzBuzz**
- **Question 2 : Cryptage / Décryptage**
